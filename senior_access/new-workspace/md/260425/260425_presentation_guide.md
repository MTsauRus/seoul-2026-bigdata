## 1막. 화두 — "이 30분은 누구의 30분인가" (슬라이드 1~4)

### 목적
심사위원이 첫 3분 안에 *"아, 이 팀은 2040 도시기본계획의 맹점을 찌르러 왔구나"* 를 알아차리게 한다.

### 슬라이드 1 — 타이틀
- **제목**: "분(分)의 격차 — 노인의 시간으로 다시 그린 서울 30분 도보일상권"
- **부제**: 2026 서울시 빅데이터 활용 경진대회 시각화 부문
- **한 줄 후크**: *"서울시 2040 도시기본계획은 '도보 30분 자족 생활권'을 약속했다. 그 30분은 누구의 30분인가."*
- 팀명·구성원

### 슬라이드 2 — 맥락 세팅 (3개의 숫자)
3개 숫자를 큼지막하게:
- **32%** — 2040년 서울 65세 이상 인구 비율 (서울시 공식)
- **0.78 m/s** — 보행보조 노인 하위 15%의 실제 보행속도
- **1.28 m/s** — 도시계획 표준 보행속도
- 하단 한 문장: *"서울시 인구 1/3의 시간은 정책 표준의 61%다."*

### 슬라이드 3 — 심재현의 "고무줄 지도"
- **자료**: `infra_별자리지도.html` / 01막 섹션 "Elastic City Map"
- **쓰는 방식**: 혜화동(인프라 상위) vs 중계본동(하위)의 반경이 속도 슬라이더에 따라 수축하는 짧은 애니메이션 캡처 3컷
- **캡션**: *"같은 동네도, 같은 30분도, 누구의 속도로 재느냐에 따라 다른 도시가 된다."*

### 슬라이드 4 — 프로젝트 선언
- **우리가 만드는 것**: "5개 생활 도메인에서 청년 30분권과 노인 30분권을 동별로 재측정한 지도 + 동네별 인프라 지문"
- **질문 다섯 개** (5도메인 = 5 질문):
  - 의료: 만성질환 관리가 동네에서 되는가?
  - 인프라: 쇼핑과 금융이 걸어서 해결되는가?
  - 복지·녹지: 일상 돌봄과 여가가 닿는가?
  - 기후: 위기 시각의 쉼터는 닿는가?
  - 교통: 동네 밖으로 나갈 수 있는가? 길은 안전한가?

## 2막. 증명 — "손실률이라는 자(尺)" (슬라이드 5~8)

### 목적
5도메인에 들어가기 전에, **팀 공통 방법론과 서울 전체 스케일의 숫자**를 먼저 박아놓는다. 이 막이 없으면 3막의 도메인별 손실률들이 맥락 없이 떠돈다.

### 슬라이드 5 — 방법론 1장 (엄밀성 과시)
- **도식**: 행정동 중심점 → OSM 보행 네트워크 스냅 → Dijkstra 30분 도달 노드 → 면적/카운트 → 손실률
- **핵심 수치**:
  - 서울 보행 그래프 **162,440 노드** (OSM 기반 전역 커버)
  - 행정동 **426개 × 2 속도 = 852회 Dijkstra 실측**
  - 유클리드 아닌 **실제 도로망 기반 거리**
- **사이드 박스**: *"유클리드 직선거리는 서울의 언덕·하천·고속도로를 무시한다. 우리는 실제로 걸을 수 있는 길만 센다."* (← 팀 차별점 선제적 공격)
- **담당**: 김성령

### 슬라이드 6 — 서울 전체 손실률 지도
- **자료**: `06_seoul_loss_map.html`을 리포트·슬라이드용 정적 캡처
- **핵심 수치**: 서울 평균 손실률 **54.7%** / 이론값 (0.88/1.28)² = 53%와 정확히 일치
- **해석 한 줄**: *"같은 30분이라도 노인은 평균적으로 서울 도로망의 45%만 커버한다."*
- **담당**: 김성령

### 슬라이드 7 — 반박 프레임: "고령화율이 문제가 아니다"
- **자료**: `07b_scatter_senior.html` 산점도
- **핵심 숫자**: 고령화율 ↔ 손실률 **r = 0.020 (p = 0.68, 무상관)** / 도로 밀도 ↔ 손실률 **r = -0.254 (p < 0.001)**
- **캡션**: *"노인이 많아서 노인이 불리한 것이 아니다. 도로가 성겨서 노인이 불리하다."*
- **왜 이 슬라이드가 중요한가**: 심사위원의 *"노인 많은 동네가 문제라는 뻔한 결론 아닌가"* 라는 반사적 의심을 미리 차단한다. 과학적 권위로.
- **담당**: 김성령

### 슬라이드 8 — "영향 노인 수" 지도
- **자료**: `07c_priority_map.html`
- **공식**: `영향 노인 수 = 손실률 × 65+ 인구 / 100`
- **TOP 5**: 노원구 상계1동(7,191명) / 양천구 신정3동(5,722) / 은평구 불광1동(5,574) / 은평구 진관동(5,467) / 중랑구 신내1동(5,407)
- **전체 합계**: *"서울 전체 약 100만 명의 65+ 노인이 보행 손실권 안에 거주한다."*
- **담당**: 김성령

## 3막. 5도메인 해부 (슬라이드 9~23)

### 목적
각 도메인이 노인 보행일상권 관점에서 **어떤 고유의 이야기**를 가지고 있는지 보여준다. 단순 지도 나열이 아니라 **도메인마다 다른 서사 장치**를 쓴다.

**공통 규칙** (모든 도메인 슬라이드):
- 첫 슬라이드: 도메인 질문 + 데이터 규모
- 둘째 슬라이드: 핵심 시각화 (도메인별 최강 자산)
- 셋째 슬라이드: **해당 도메인 취약 TOP 5동 + 영향 노인 수**

### D1. 교통·이동 (슬라이드 9~10) — *유호준 파트 완성 후 확정*
슬롯은 예약, 내용은 유호준 산출물 수령 후 확정.

### D2. 생활 인프라 (슬라이드 11~13) — 담당: 심재현
- **11. 질문과 데이터**: *"노인이 일상 쇼핑과 금융을 걸어서 해결할 수 있는가?"* / 전통시장 195 + 슈퍼 31,024 + 은행 지점 시계열
- **12. 핵심 시각화 — "사라지는 그림자"**: 별자리지도 03막. 서울 시중은행 점포 2000→2024, **58.8% 감소 / 오프라인 의존 노인 340만**. 시계열 + 은유 조합은 팀 내 유일.
- **13. 인프라 취약 TOP 5동**: *동 단위 재집계 후 확정 — 심재현 스프린트 과제*

### D3. 의료 (슬라이드 14~16) — 담당: 이정태
- **14. 질문과 데이터**: *"노인이 만성질환 관리를 동네에서 할 수 있는가?"* / 병의원 12,154 + 약국 5,808
- **15. 핵심 시각화 — 청년×노인 산점도**: `02_access_scatter.html`. y=x 기준선 + (0.65)² 이론선. 팀 내 가장 지적인 한 장.
  - *"점이 회색 대각선(손실 0) 아래로 내려갈수록, 주황 점선(물리적 이론치) 아래로 내려갈수록, 그 동네는 구조적으로 더 불평등하다."*
- **16. 의료 취약 TOP 5동**: *OSM 엔진 이식 재계산 후 확정 — 이정태 스프린트 과제*

### D4. 복지·녹지 (슬라이드 17~19) — 담당: 양석준
- **17. 질문과 데이터**: *"노인의 일상 돌봄과 여가가 걸어서 가능한가?"* / 노인여가복지시설 + 서울 공원
- **18. 핵심 시각화 — 복지 × 녹지 사분면 산점도**: `welfare_green_scatter.png`. 중앙값 십자선으로 4사분면 해석. 동 단위 426개 점으로 재생성 후 클러스터 드러내기.
  - *"복지 낮음 × 녹지 낮음 = 최악 사분면" 위치에 있는 동들이 이중으로 취약하다.*
- **19. 복지·녹지 취약 TOP 5동**: *속도·지오코딩·필터 수정 후 확정 — 양석준 스프린트 과제*

### D5. 기후 재난 (슬라이드 20~23) — 담당: 김성령
- **20. 질문과 데이터**: *"위기 시각의 쉼터는 닿는가?"* / 폭염쉼터 4,107 + 한파쉼터 1,642 + 제설함 10,437 + 열선
- **21. 핵심 시각화 ① — "격차의 계절 역전"**: 폭염쉼터 노인 15분 접근 **93.4%** vs 한파쉼터 **89.2%**. 같은 서울인데 *"추울수록 닿기 어렵다."*
- **22. 핵심 시각화 ② — 결빙 취약 지도 v3**: `10_b3_snow_icing_v3_260421.html`. 한강·산지 제외 도심 431.9 km² 기준 **200m 제설함 커버 33%만 · 67%가 사각지대**. 열선 레이어와 겹쳐 "**인지했지만 방치된 구간**" 판별.
- **23. 기후 취약 TOP 5동**: *김성령이 기존 데이터에서 바로 산출 가능*

## 4막. 통합 — "동네별 인프라 지문" (슬라이드 24~26)

### 목적
5개 도메인의 결과를 **하나의 장면**으로 합친다. 브리프가 처음부터 약속한 메인 시각화.

### 슬라이드 24 — 레이더 차트 하나 (개념 학습용)
- 대표 1개 동(예: 노원구 상계1동, 영향 노인 1위)의 5축 레이더:
  - 교통·인프라·의료·복지·기후 각 축에 0~100 접근 점수
- 이 동의 "모양"이 어떻게 생겼는지 설명
- 캡션: *"우리는 이 모양을 '인프라 지문(Fingerprint)'이라 부른다. 모든 동네는 서로 다른 지문을 가진다."*

### 슬라이드 25 — Small Multiples: 서울의 25개 지문
- **자료 (신규 제작)**: 각 구 대표동 1개씩 25개 레이더 + 서울 평균 1개 = **26 small multiples**
- 모양별 그룹 분류 (3~4 타입):
  - **A 타입** (사각형에 가까움): 전반적으로 접근성 양호. 예: 종로구 일부, 강남구 일부
  - **B 타입** (한쪽이 길쭉함): 특정 도메인만 강점. 예: 서초구 = 의료 특화
  - **C 타입** (전반적으로 작음): 전방위 취약. 예: 노원구 상계1동 = 복지·녹지·기후 동시 약함
  - **D 타입** (중심 공백): 접근 자체가 어려운 동. 예: 은평구 진관동 = 외곽 저밀도
- 각 타입별로 **서로 다른 정책 개입**이 필요함을 시사
- **담당**: 심재현(디자인) + 김성령(데이터) — 신규 공동 작업

### 슬라이드 26 — 통합 손실 서울 지도
- 5도메인 손실률을 가중 평균한 **"생활권 종합 손실 지수"** 코로플레스 1장
- 영향 노인 수 기준 **전 도메인 종합 TOP 10동** 하이라이트
- 통합 대시보드 랜딩 페이지 역할

## 5막. 킥 — 하나의 장면으로 응축 (슬라이드 27~28)

### 목적
심사위원이 **3일 뒤에도 기억할** 한 장면을 만든다. 수상작들을 보면 "가독성은 별로지만 이상한 모양의 참신한 그래프가 꼭 있다" 는 관찰에 정면으로 응답.

**후보 3개를 제시한다. 팀에서 한두 개 선택.**

### 🎬 킥 후보 A — "김복자 씨의 월요일" (강력 추천)

**개념**: 77세 가상 페르소나 김복자 씨(노원구 상계1동 거주 · 영향 노인 1위 동)가 하루를 보낸다. 5개 도메인을 페르소나 한 명의 동선으로 연결.

**구성**:
```
08:00  아침 약국 방문      — D3 의료      (노인 9분 vs 청년 5분)
09:30  경로당 가기          — D4 복지      (도달 가능)
11:00  전통시장 장보기      — D2 인프라    (청년 10분 거리, 노인 17분)
13:30  은행 ATM 방문        — D2 인프라    (걸어서 도달 불가. 버스 필요)
15:00  폭염쉼터 대피        — D5 기후      (도달 가능)
17:00  지하철역 외출        — D1 교통      (엘리베이터 없는 역, 포기)
19:00  귀가
```

**시각화**: 상계1동 지도 위에 김복자 씨의 동선을 시간별로 그림. 각 시점마다 청년 vs 노인 소요 시간·도달 가능성 대비. **도달 불가 목적지는 빨갛게 × 표시.**

**파워**: 숫자 100만 명이 한 사람의 하루로 응축된다. 심사위원이 뇌에 새길 이름 하나가 생긴다.

**제작**: 심재현 디자인 + 김성령 데이터 (동선·시간 계산)

### 🎬 킥 후보 B — "두 개의 서울, 같은 30분"

**개념**: 서울 지도 두 장을 나란히 놓는다. 왼쪽은 **"청년의 30분권"**, 오른쪽은 **"노인의 30분권"**. 완전히 같은 시각·같은 정책(2040 도보 30분 자족 생활권)이지만, 두 지도의 커버 면적이 육안으로도 40% 차이난다.

**시각화**: 스플릿 뷰. 색 채워진 영역 = 30분 내 5도메인 모두 접근 가능한 지역. 왼쪽은 서울 거의 전체가 파랗게, 오른쪽은 동심원처럼 구멍이 뚫림.

**임팩트 문구**: *"같은 도시, 같은 정책, 다른 현실. 이 두 장이 우리 프로젝트의 전부다."*

**파워**: 단순함. 한 장(정확히는 두 장)으로 프로젝트 전체 메시지 끝.

**제작**: 김성령 단독 가능 (기존 데이터 재구성)

### 🎬 킥 후보 C — "서울 30분의 재측정" 인터랙티브

**개념**: 웹 대시보드 안에서 관객(심사위원)이 직접 슬라이더를 움직이며 속도를 바꾼다. 1.28 → 1.12 → 0.88 → 0.78로 내려갈 때마다 서울 전체 도메인 커버리지가 실시간으로 쪼그라든다.

**시각화**: 심재현의 "고무줄 지도" 개념을 **5도메인 × 서울 전체**로 확장. 슬라이더 하나로 5개 지도가 동시에 수축.

**임팩트**: 심사위원이 손으로 직접 노인의 시간을 체험. 대시보드의 핵심 기능으로 배치.

**파워**: 인터랙티브. 최종 경연(시청 다목적홀 라이브 시연) 에서 즉각 먹힘.

**제작**: 심재현 + 김성령 (대시보드 통합 작업의 부산물로 가능)

## 6막. 제언 (슬라이드 29~30)

### 슬라이드 29 — 정책 제언 3개

노인 관점에서의 "도보일상권" 을 정책으로 옮기기 위한 **구체적 3개 제언**. 추상어 없이.

1. **"30분의 재정의"**
   - 2040 도시기본계획의 "도보 30분 자족 생활권" 기준을 **보행속도 0.88 m/s (노인 평균)** 기준으로 재작성.
   - 기존 2,304m 반경 → **1,584m 반경**으로 읽는 정책 표준을 병기.

2. **"동 단위 인프라 지문(Fingerprint) 모니터링 도입"**
   - 동별 5도메인 접근 점수를 매년 갱신하는 **서울시 공식 지표**로 등록.
   - 취약 도메인 조합(Fingerprint 타입)별 차등 개입.

3. **"영향 노인 수 기반 예산 우선순위 배정"**
   - 손실률 단독이 아닌 **`손실률 × 65+ 인구`** 지표로 복지·보건·교통 예산 배분.
   - 본 프로젝트 TOP 10동을 **2027년 시범 개입 지역**으로 지정 제안.
